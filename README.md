このリポジトリでは､Zenn の記事を管理しています｡

## 📋️ How to use

`zenn-cli`を使って記事の管理を行うことができます｡

**記事の作成**

```bash
$ pnpm zenn new:article
```

**本の作成**

```bash
$ pnpm zenn new:book
```

**記事のプレビュー**

```bash
$ pnpm zenn preview
```
